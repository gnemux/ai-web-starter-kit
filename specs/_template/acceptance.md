# Acceptance

## Functional Checks

- [ ] Main user path works.
- [ ] Empty state works.
- [ ] Loading state works.
- [ ] Error state works.
- [ ] Permission state works when applicable.

## Technical Checks

- [ ] Typecheck passes.
- [ ] Lint passes.
- [ ] Build passes.
- [ ] No secrets are committed.
- [ ] Relevant docs are updated.

## Product Checks

- [ ] The feature matches the product spec.
- [ ] The UI is usable without reading implementation notes.
- [ ] Success metrics or events are defined.
